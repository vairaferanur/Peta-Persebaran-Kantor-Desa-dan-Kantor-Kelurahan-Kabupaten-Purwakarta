# Peta-Persebaran-Kantor-Desa-dan-Kantor-Kelurahan-Kabupaten-Purwakarta
Peta persebaran lokasi kantor desa dan kantor kelurahan di Kabupaten Purwakarta Tahun 2024

## Deskripsi
Repository ini hasil kegiatan magang yang dilaksanakan di bawah koordinasi Dinas Komunikasi dan Informatika Kabupaten Purwakarta, dengan penugasan di Dinas Pemberdayaan Masyarakat dan Desa Kabupaten Purwakarta. Peta persebaran kantor desa dan kantor kelurahan disusun untuk menampilkan sebaran lokasi fasilitas pemerintahan tingkat desa dan kelurahan di Kabupaten Purwakarta. Peta ini berfungsi sebagai media visualisasi informasi lokasi kantor desa dan kelurahan secara spasial sebagai informasi umum wilayah.

## Informasi yang Disajikan
- Lokasi kantor desa dan kantor kelurahan
- Batas administrasi wilayah

## Tools
- QGIS
- ArcGIS
- Google Earth
- Microsoft Excel

## Output
- Peta Persebaran Kantor Desa dan Kantor Kelurahan Kabupaten Purwakarta Tahun 2024
